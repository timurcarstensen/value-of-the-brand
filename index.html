Brand Values Visualized

We compiled a unique database that includes more than 43,000 financial brand values that were obtained from the publicly available websites of four different brand-valuation providers. The database allows for a diverse comparison of different financial brand values: For example, visualization of time trends in financial brand values; comparison of different brand values in different industries; comparison of financial brand values from different providers for the same brand etc.. You can customize the comparisons as you like.
